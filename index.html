Color Golf

Welcome to Color Golf! This game tests your RGB color skills. Your goal is to match the target color (on the right) by adjusting the RGB values of the current color (on the left). You write changes to the Red, Green, and Blue values of the color. As usual with RGB color, the values are between 0 and 255. Your changes can be positive or negative. Enter the changes you want to make and see how close you can get to the target color. Try to match the color in as few moves as possible. Choose “Daily Game” to play today’s game, or “Random Game” to play a randomly generated challenge.
